Este tutorial foi escrito para quem nunca usou o deepH Plus antes.

Sempre que você ver um texto assim:
  [CAIXA DE DESTAQUE] → É uma dica ou aviso importante.

  .\deeph.exe ui

iniciar ferramentas manuais, caso não queira usar o atalho.

Vamos começar!


==============================================================
O QUE É O deepH Plus?
==============================================================

O deepH Plus é o seu "escritório de assistentes virtuais" pessoal,
agora com uma interface moderna, premium, escura (Dark Theme) e
muito intuitiva baseada em conceitos de design avançados.

Imagine que você tem vários "Agentes" que moram dentro
do seu computador. Cada um desses ajudantes tem um nome,
uma especialidade e habilidades (skills).

Você pode pedir para eles:
  - Responder perguntas complexas
  - Analisar textos e documentos (com a skill de arquivo)
  - Pesquisar na internet
  - Resolver problemas lógicos e escrever códigos

A GRANDE NOVIDADE: A nova versão possui uma Dashboard Visual
interativa, com relatórios em tempo real (Sparklines) e um Chat
com layout sofisticado. Tudo rodando localmente na sua máquina.


==============================================================
PARTE 1: ABRINDO O deepH Plus PELA PRIMEIRA VEZ
==============================================================

Siga estes passos na ordem. São simples!

--------------------------------------------------------------
PASSO 1 — Encontre a pasta do deepH Plus

Abra o "Explorador de Arquivos" do Windows. 

Navegue até a seguinte pasta:
  C:\Users\BOSS\deepH

[DICA] Se você não souber como chegar lá, clique na barra
de endereços no topo do Explorador e cole o caminho acima.

--------------------------------------------------------------
PASSO 2 — Procure o arquivo start_ui.bat

Dentro da pasta, você vai encontrar vários arquivos.
Procure um com o nome exato:

  start_ui.bat

--------------------------------------------------------------
PASSO 3 — Abra o Painel dando dois cliques

Dê um DUPLO CLIQUE em start_ui.bat.

O que vai acontecer:
  1. Uma janela preta (terminal) vai abrir e iniciar o motor em Go.
  2. Em seguida, o seu navegador (Chrome, Edge etc) vai abrir
     automaticamente na página http://localhost:7730.
  3. A elegante TELA DO PAINEL DEEPH PLUS vai carregar, exibindo
     um visual moderno com tons verdes neon (Neural Link Active).

[AVISO IMPORTANTE]: Não feche a janela preta! Ela é o
cérebro (back-end). Se fechar, o painel Front-end para de responder.
Pode minimizá-la tranquilamente.


==============================================================
PARTE 2: CONHECENDO O NOVO PAINEL VISUAL
==============================================================

--------------------------------------------------------------
2.1 — A BARRA LATERAL ESQUERDA (Navegação Rápida)

Nela você encontra botões minimalistas usando ícones:
  - QUADRADOS (Dashboard) → Volta à tela inicial com todos os agentes.
  - SINAL DE + (Add)      → Cria um novo agente rapidamente.
  - ENGRENAGEM (Settings) → Gerencia suas configurações e API Keys.

--------------------------------------------------------------
2.2 — TELA INICIAL (DASHBOARD)

A primeira tela que você vê é o Dashboard. 
Lá aparecem CARTÕES translúcidos e vívidos de agentes.

Cada cartão mostra:
  - O NOME do agente e seu Provedor (ex: DeepSeek ou OpenAI)
  - A DESCRIÇÃO: o que ele faz.
  - SINAIS DE ATIVIDADE: Você verá pílulas coloridas informando
    se a API está ativa.
  - Clicar num Cartão abre a tela de Chat.
  - Clicar na ferramenta perto do nome dele abre as Configurações (Editor).


==============================================================
PARTE 3: CONFIGURANDO AS CHAVES DE API
==============================================================

Para o deepH Plus ter Inteligência, ele precisa de uma chave 
de acesso aos cérebros de IA da nuvem (ou um local).

--------------------------------------------------------------
PASSO 1 — Acesse a Engrenagem (Settings)

No menu esquerdo, clique no ícone de engrenagem.
A tela de "System Configuration" vai aparecer com seus KPI (Estatísticas
simuladas) e os "Active Keys".

--------------------------------------------------------------
PASSO 2 — Cole as chaves nos cards de segurança

Você verá Provedores já listados em layouts sofisticados:
  - DeepSeek (Principal)
  - OpenAI (Ativo)
  - Anthropic (Em espera)

Nos campos escuros de input (sk-proj...), cole sua chave respectiva.
Exemplo: crie uma chave no platform.deepseek.com e cole no campo
do DeepSeek.

--------------------------------------------------------------
PASSO 3 — Salve e Confirme

Clique em "Save Settings" no botão verde neon brilhante no final.
Ele gravará tudo localmente no seu arquivo escondido `config.json`.
A chave NUNCA é enviada para ninguém a não ser a central do modelo!


==============================================================
PARTE 4: CONVERSANDO COM SEUS AGENTES (O CHAT)
==============================================================

1. Volte na tela inicial (clicando no primeiro botão do menu).
2. Clique no seu agente desejado (ex: "guide").
3. A nova janela de chat se abrirá simulando um "Neural Link".
4. Abaixo, você tem uma barra translúcida. Digite como em mensageiros.
5. Aperte a Seta Para Cima (Enviar) ou pressione Enter.
6. A resposta vai jorrar magicamente na tela em tempo real,
   com destaques visuais ricos e formatação atraente!


==============================================================
PARTE 5: O EDITOR DE AGENTES DE ALTA PERFORMANCE
==============================================================

Apertando em "+ " na barra lateral ou na engrenagem de um agente
específico na Dashboard, você abre o EDITOR DE AGENTE.

Ele permite ajustar o agente de 2 formas: VISUAL ou YAML!
(Na nova versão, a tab do YAML é exibida de forma segura.)

No Visual você pode:
- Mudar Nome, Descrição e o System Prompt (Regras dele).
- Escolher qual Modelo e Provider ele responde (OpenAI, DeepSeek, Ollama).
- Clicar nos cards em grade para ATIVAR SKILLS (ex: "Code Interpreter", 
  "Web Search", "Memory"). As skills selecionadas brilham em verde!

Tudo que você trocar é gravado como mágica no arquivo .yaml local no disco.
Clique em "Deploy Agent" (salvar).


==============================================================
PARTE 6: MODO TERMINAL (AVANÇADO)
==============================================================

O DeepH Plus continua tendo o DNA robusto para desenvolvedores.
Abra um Powershell na pasta e rande:

  .\deeph.exe run "agente1" "Sua tarefa aqui"

Você também pode parear agentes para trabalharem juntos no terminal!
Mas agora, você também pode curtir todo o fluxo via Navegador com luxo e estilo.


==============================================================
SOLUCIONANDO PROBLEMAS
==============================================================

PROBLEMA: "Failed to fetch"
A janelinha preta fechou! Lembre-se, o backend Go precisa rodar para o UI funcionar. Abra o `start_ui.bat` de novo.

PROBLEMA: Erro nas respostas do Chat.
Pode ser que a API Key esteja vencida, incorreta ou sem limite de custos.
Vá nas Configurações, revise as chaves gravadas.
